## Installation

Clone the repo

```bash
git clone https://github.com/kanhaiya04/Tech-blogs.git
cd Tech-blogs/
```

Use the package manager to install dependencies.

```bash
npm install
```

## Run on local server

To start the server
```bash
node app
```
